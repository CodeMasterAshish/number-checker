# Number Checker

A simple JavaScript project that checks the properties of a number entered by the user.

## Features

- Detects whether a number is Positive, Negative, or Zero
- Checks if a number is Even or Odd
- Displays the result instantly
- Beginner-friendly JavaScript project

## Technologies Used

- HTML5
- JavaScript

## Project Structure

number-checker/
│
├── index.html
├── script.js
└── README.md

## How to Run

1. Download or clone the project.
2. Open `index.html` in your browser.
3. Enter a number.
4. Click the **Check Number** button.
5. View the result.
